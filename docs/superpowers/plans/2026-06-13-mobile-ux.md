# Mobile / Touch UX Pass тАФ Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the PWA genuinely playable on a phone тАФ responsive shell + deck + per-panel layouts + touch targets тАФ with no save/runtime changes.

**Architecture:** Pure CSS `@media` work plus one guarded `scrollIntoView` `$effect`. Two breakpoints: **тЙд 720px** (mobile reflow) and **тЙд 480px** (hide decorative Pixi). Mobile scroll lives in inner containers (body stays `overflow:hidden`), so the masthead + nav pin and content scrolls. Desktop layout is untouched тАФ every new rule lives inside a `max-width` media block.

**Tech Stack:** Svelte 5, CSS, Vite/PWA. No tests beyond build/check + existing render-smokes (responsive CSS isn't unit-testable).

**Spec:** [docs/superpowers/specs/2026-06-13-mobile-ux-design.md](../specs/2026-06-13-mobile-ux-design.md)

**Conventions:** use app.css design tokens; **no AI co-author trailer**; docs updated same pass. Verify with `npm run check`, `npm test`, `npm run build`. **Per-task: also eyeball at a narrow width in the browser** (the real signal for this work).

## Task 2: GameLayout shell тАФ the core reflow

**Files:** `src/ui/GameLayout.svelte`

The shell is flex-based: `.masthead` (brand ┬╖ `.view-nav` ┬╖ `.mast-right`), `.dial-bar` (already `overflow-x:auto`), `.content` (`display:flex`: `.viewport-col` 236px ┬╖ `.stage-col` flex:1 ┬╖ `.engine-col` 256px).

- [ ] **Step 1: Add the active-nav auto-scroll effect**

In the `<script>`, after the `view`/`canSee*` declarations, add:
```ts
  let navEl = $state<HTMLElement | null>(null)
  $effect(() => {
    // re-runs when `view` changes; centre the active nav button in the scroll strip
    void view
    navEl?.querySelector('.view-btn.active')?.scrollIntoView({ inline: 'center', block: 'nearest', behavior: 'smooth' })
  })
```
And bind the nav element тАФ change `<nav class="view-nav">` to:
```svelte
    <nav class="view-nav" bind:this={navEl}>
```

- [ ] **Step 2: Add the mobile media block**

Append to the GameLayout `<style>` (end, before `</style>`):
```css
  /* тФАтФА Mobile (тЙд 720px) тФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФА */
  @media (max-width: 720px) {
    /* slim masthead; nav wraps to its own full-width scroll strip */
    .masthead { flex-wrap: wrap; padding: 6px 12px; padding-top: max(6px, var(--safe-top)); row-gap: 6px; }
    .brand-cog { width: 26px; height: 26px; }
    .brand-sub { display: none; }
    .brand { order: 0; }
    .mast-right { order: 1; }
    .view-nav {
      order: 2; width: 100%; overflow-x: auto; overflow-y: hidden;
      border-radius: 4px; scrollbar-width: none; -webkit-overflow-scrolling: touch;
    }
    .view-nav::-webkit-scrollbar { display: none; }
    .view-btn { padding: 0 14px; min-height: var(--tap-min); white-space: nowrap; flex: 0 0 auto; }
    .fr-label { display: none; }
    .fortune-readout { padding: 4px 10px; }

    /* dials: keep the existing horizontal scroll, size for touch */
    .dial { min-width: 76px; min-height: var(--tap-min); padding: 6px 12px; flex: 0 0 auto; }

    /* deck: single scrollable column тАФ StagePanel тЖТ Engine тЖТ slim Pixi strip */
    .content { flex-direction: column; overflow-y: auto; overflow-x: hidden; -webkit-overflow-scrolling: touch; padding-bottom: var(--safe-bottom); }
    .stage-col  { order: 0; flex: 0 0 auto; overflow-y: visible; }
    .engine-col { order: 1; width: 100%; flex: 0 0 auto; border-left: none; border-top: 1px solid var(--brass-deep); overflow-y: visible; }
    .viewport-col { order: 2; width: 100%; height: 120px; flex: 0 0 auto; }
  }

  /* тФАтФА Compact (тЙд 480px): drop the decorative Pixi viewport тФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФАтФА */

    .viewport-col { display: none; }
  }
```
